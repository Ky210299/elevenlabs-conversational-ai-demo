# Demo approach
This demo is a simple chatbot that uses ElevenLabs Conversational AI API and a interactive avatar syncronized with the ElevenLabs response.

# Demo Features:
- Streaming through WebSocket
- Real-time responses
- Interactive avatar synchronization
